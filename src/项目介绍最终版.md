The low-code visualization low-code platform is my most representive project. An online digital visualization application development platform built on React, users can create visually data visualization applications (large screens) through a graphical interface, without design experience or technical background.

This project is based on large-screen component libraries such as Echarts and DataV. It constructs dynamic form configuration visual components through JSON Schema, packages them through a component production factory, generates platform-compatible versions, and uploads them to the component development list. Users can build large-screen applications by dragging and dropping components or editing the canvas. After preview and publish, through a cross-screen linkage solution based on the WebSocket protocol enables data communication between large screens, breaking different terminal limitations.
